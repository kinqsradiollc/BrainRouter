# @brainrouter/types

Shared TypeScript types for the [BrainRouter](https://github.com/kinqsradiollc/BrainRouter) monorepo — used by `brainrouter` (CLI), `@brainrouter/mcp-server`, and `@brainrouter/sdk`.

No runtime code, just types.

## Install

```bash
npm install --save-dev @brainrouter/types
```
